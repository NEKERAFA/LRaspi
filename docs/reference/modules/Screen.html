Functions

screen.init ()
Initializes the screen subsystem. The interpreter calls this function internally, so should not be used explicitly.
screen.clear (color)
Clears the screen with an specified color

Parameters:

  • color color A color object
screen.update ()
Updates the screen and returns if the program was running

Returns:

    boolean true if the program was running, false otherwise

Usage:

    local quit = false
while not quit do
   -- updating screen
   quit = screen.update()
end
screen.getcanvas ()
Gets the current canvas

Sets the render target to a canvas

Parameters:

  • A canvas canvas object

Resets the render target to the screen
screen.blit (texture, x, y)
Blits an texture onto the screen

Parameters:

  • texture texture A texture object to blit
  • x int The x-axis position to draw the texture
  • y int The y-axis position to draw the texture
screen.print (x, y, textstring, A)
Prints a text onto the screen

Parameters:

  • x int The x-axis position to draw the texture
  • y int The y-axis position to draw the texture
  • textstring string The text to print
  • A color color object
