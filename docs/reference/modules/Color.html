color.getpalette ()
Gets a list of 15 color to use.

Returns:

    table A list with the following colors:
    • white (255, 255, 255)
    • black (0, 0, 0)
    • red (255, 0, 0)
    • green (0, 255, 0)
    • blue (0, 0, 255)
    • yellow (255, 255, 0)
    • cyan (0, 255, 255)
    • magenta (255, 0, 255)
    • marron (128, 0, 0)
    • grass (0, 128, 0)
    • navy (0, 0, 128)
    • olive (128, 128, 0)
    • turquoise (0, 128, 128)
    • violet (128, 0, 128)
    • gray (128, 128, 128)
