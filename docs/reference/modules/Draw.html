draw.ellipse (mode, x, y, radiusx, radiusy, color, segments)
Draws an ellipse

Parameters:

  • mode string A string that controls whether shapes are draw, as the following values:
    • fill for draw filled shape
    • outline for draw outlined shape
  • x number The position of the center along the x-axis
  • y number The position of the center along the y-axis
  • radiusx number The radius of the ellipse along the x-axis (a half of the ellipse width)
  • radiusy number The radius of the ellipse along the y-axis (a half of the ellipse height)
  • color color The color of the ellipse
  • segments number The number of segments used for drawing the ellipse

draw.arc (mode, type, x, y, radiusx, radiusy, angle1, angle2, color, segments)
Draws an ellipse arc

Parameters:

  • mode string A string that controls whether shapes are draw, as the following values:
    • fill for draw filled shape
    • outline for draw outlined shape
  • type string A string that controls the type of arc, as the following values:
    • pie for draw the arc as a slice of pie, connecting the end-points with the center
    • open for draw the arc without the end-points unconnected
    • closed for draw the arc with the end-points connected to each other
  • x number The position of the center along the x-axis
  • y number The position of the center along the y-axis
  • radiusx number The radius of the ellipse along the x-axis (a half of the ellipse width)
  • radiusy number The radius of the ellipse along the y-axis (a half of the ellipse height)
  • angle1 number The angle at which the arc starts
  • angle2 number The angle at which the arc ends
  • color color The color of the arc
  • segments number The number of segments used for drawing the arc
